Karma Frontend es una aplicación web desarrollada en React para la gestión de un negocio de joyería y accesorios. La aplicación ofrece funcionalidades para administrar inventario, ventas, generación de etiquetas, carrito de compras y procesamiento de pagos.

Tecnologías Utilizadas
React: Biblioteca principal para la construcción de la interfaz
React Bootstrap: Componentes de UI responsivos
CSS Custom: Estilos personalizados para mejorar la experiencia de usuario
Font Awesome: Iconografía moderna y profesional
HTML2Canvas/jsPDF: Generación de PDFs para etiquetas e impresión
RESTful API: Comunicación con el backend a través de API
